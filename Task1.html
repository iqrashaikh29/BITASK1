Drowsiness Detection System using ML

This proposed system describes a machine learning approach for drowsiness detection. Face detection is employed to locate the regions of the driver's eyes,
which are used as the templates for eye tracking in subsequent frames. Finally, the tracked eye's images are used for
drowsiness detection in order to generate warning alarms. This proposed approach has three stages: detecting Face,
detecting Eyes and detecting drowsiness. Image processing is used to recognize the face of the driver and then its extracts the
image of the eyes of the driver for detection of drowsiness. The HOG + Linear SVM algorithm takes as captured frames of image
and then the detected face is considered as output. Next, Contourfitting is used for tracking eyes from the detected face. Using EAR (Eye Aspect Ratio) the eye state is detected.
The system uses frames for face and eye-tracking, and the average correct rate for eye location and tracking could achieve 95.0% based on some test videos. Thus, the proposed approach for real-time drivers drowsiness detection is a low-cost and effective solution method
